## Quick Setup Guide

Get Ignotus running on your desktop in just a few minutes.

### Prerequisites

Before installing Ignotus, make sure you have:

- **macOS 13.0+** (required for latest features)
- **Xcode 14.0+** for building the application
- **8GB RAM** minimum (16GB recommended)
- **2GB free disk space**
- **Internet connection** for initial setup and API access

### Step 1: Clone and Build

<Card
  title="Clone Repository"
  icon="github"
  href="https://github.com/ignotus-ai/ignotus-swift"
>
  Clone the Ignotus Swift repository from GitHub
</Card>

1. **Clone the repository**
   ```bash
   git clone https://github.com/ignotus-ai/ignotus-swift.git
   cd ignotus-swift
   ```

2. **Open in Xcode**
   - Open `Ignotus.xcodeproj` in Xcode 14.0 or later
   - Select your target device (Mac)
   - Build the project (Cmd+B)

3. **Run the application**
   - Press Cmd+R to run Ignotus
   - The app will launch and guide you through initial setup

### Step 2: Initial Configuration

When you first launch Ignotus, you'll be guided through:

1. **Permissions Setup**
   - Grant screen recording access for Live Mode
   - Allow accessibility permissions for enhanced features
   - Enable microphone access for voice commands
   - Grant access to Adobe Acrobat Pro for PDF processing

2. **API Configuration**
   - Configure Google Gemini API for AI processing
   - Set up API keys in the settings
   - Choose your preferred AI model (light, medium, pro)

3. **ChromaDB Setup** (Optional)
   - Install ChromaDB for knowledge management
   - Configure local ChromaDB server
   - Set up initial knowledge base

### Step 4: Advanced Features

#### Inference Graph System
Ignotus includes a powerful inference graph system for complex workflows:

- **Dynamic Graph Generation**: LLM creates workflows automatically
- **PDF Form Processing**: Fill forms with intelligent data extraction
- **Multi-step Workflows**: Complex tasks broken into manageable steps
- **Real-time Feedback**: See each step as it executes

#### Knowledge Management
Set up your personal knowledge base:

1. **Install ChromaDB**
   ```bash
   pip install chromadb
   ```

2. **Start ChromaDB Server**
   ```bash
   chroma run --host localhost --port 8000
   ```

3. **Add Contexts**
   - Use the "Add Context" feature in Ignotus
   - Upload documents and create knowledge embeddings
   - Query your knowledge base for relevant information

### Global Shortcuts

Ignotus provides several global shortcuts for quick access:

- **Opt+Space**: Capture and analyze current window
- **Opt+U**: Email reply assistance
- **Opt+P**: Toggle window focus tracking
- **Opt+T**: Open To-Do list
- **Opt+S**: Open settings
