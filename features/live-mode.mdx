## Live Mode

Ignotus's Live Mode is your AI co-pilot that watches your screen and provides automatic assistance without you having to ask.

### What is Live Mode?

Live Mode is Ignotus's most powerful feature - it continuously monitors your desktop activity and provides intelligent, contextual assistance. Think of it as having an AI assistant that's always watching and ready to help.

### How to Activate Live Mode

1. **Keyboard Shortcut**: Press `Cmd/Ctrl + Shift + I`
2. **Menu Option**: Right-click the Ignotus icon and select "Enable Live Mode"
3. **Settings**: Go to Preferences → Live Mode → Enable

### Live Mode in Action

#### Email Assistance
When you're composing an email, Live Mode can:
- Suggest better subject lines
- Recommend tone adjustments
- Identify missing information
- Propose follow-up actions

#### Document Processing
While working with documents, Live Mode:
- Extracts key information automatically
- Identifies action items and deadlines
- Suggests related documents
- Highlights important sections

#### Task Management
Live Mode helps with task organization by:
- Creating to-dos from conversations
- Setting reminders for deadlines
- Grouping related tasks
- Prioritizing your workload

### Privacy & Security

<Note>
Live Mode processes information locally on your machine. Your screen data never leaves your device unless you explicitly choose to share it.
</Note>

### Customization Options

You can customize Live Mode to suit your preferences:

- **Sensitivity Level**: Adjust how frequently Ignotus provides suggestions
- **Notification Style**: Choose between subtle hints or detailed recommendations
- **Focus Areas**: Specify which types of content to monitor (emails, documents, etc.)
- **Quiet Hours**: Set times when Live Mode should be less active

### Troubleshooting Live Mode

**Live Mode not working?**
- Check that screen recording permissions are granted
- Ensure Ignotus has accessibility access
- Restart Ignotus if the feature becomes unresponsive

**Too many notifications?**
- Adjust the sensitivity level in settings
- Set up quiet hours for focused work time
- Customize which types of suggestions you want to see

### Advanced Live Mode Features

#### Smart Context Switching
Live Mode understands when you switch between different projects and adjusts its assistance accordingly.

#### Learning Your Patterns
Over time, Live Mode learns your work patterns and provides more personalized assistance.

#### Integration with Other Tools
Live Mode works seamlessly with:
- Email clients (Outlook, Gmail, Apple Mail)
